Simple Animated Counter with JS

Simple Animated Counter with JS is project created in just one night to test and learn new skills in JavaScript. It uses all native and Vanilla JavaScript code writter by ES2017 standards. It is simple counter upwards without using any other library and design is basic CSS Grid and Flexbox. All data is collected from Idzan Marko's YouTube Channel and it will be updated on monthly basis.

Technologies used: JavaScript, HTML, CSS, SASS
